# c-embedded-foundations
A collection of C programming exercises aimed at building strong foundations for embedded and firmware development. Includes time conversion, mathematical computations, structures, pointers, and memory-aware coding practices, written with an embedded-systems mindset.

# Topics Covered
- Data types and type promotion
- Functions and control flow
- Structures
- Pointers and memory concepts
- Embedded-style problem solving

# Purpose
This repository is part of a structured training path toward professional
firmware development, including ESP32, ESP-IDF, and FreeRTOS.
